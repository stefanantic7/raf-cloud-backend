# RAF Cloud (backend)

RAF Cloud is a simulation of the system for managing cloud computers.

Application is based on RESTful API and connected with frontend:
[RAF Cloud frontend](https://github.com/stefanantic7/raf-cloud-frontend) 

Authentication is based on JWT generated by the server.

## Database configuration
The project is based on a relational database so 
you should configure your database credentials into `application.properties` file.

Property `spring.jpa.hibernate.ddl-auto` is set to `auto` so all tables will be automatically generated.

```
mvn clean install 
```

## Install dependencies
```
mvn clean install 
```
